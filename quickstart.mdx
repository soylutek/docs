---
title: "Get Started with BMP Manipulator in 5 Minutes"
sidebarTitle: "Quickstart"
description: "Learn how to inspect a BMP file, add metadata, and hide a secret message using BMP Manipulator's CLI and Python API in under 5 minutes."
---

By the end of this guide, you will have installed BMP Manipulator, inspected a real BMP file's header and metadata, embedded custom key-value metadata into it, and hidden a secret text message using LSB steganography — all from both the CLI and the Python API.

<Steps>

  <Step title="Install">
    Clone the repository and install the required dependencies:

    ```bash
    git clone https://github.com/soyluoltu/bmp-manipulations.git
    cd bmp-manipulations
    pip install -r requirements.txt
    ```
  </Step>

  <Step title="Inspect a BMP File">
    Run the `info` command to view the header details and metadata status of any BMP file:

    ```bash
    python bmp_manipulator.py info photo.bmp
    ```

    You should see output similar to:

    ```
    File: photo.bmp
    Size: 1,382,454 bytes
    Header Type: BITMAPINFOHEADER
    Dimensions: 800 x 576
    Bit Depth: 24
    Compression: BI_RGB
    Metadata: None
    ```

    This confirms the file's dimensions, colour depth, compression method, and whether any metadata is already embedded.
  </Step>

  <Step title="Add Metadata">
    Use the `metadata add` command to embed a key-value pair into your BMP file:

    ```bash
    python bmp_manipulator.py metadata add photo.bmp \
      --key "Author" --value "Alice" \
      --output photo_meta.bmp
    ```

    Verify the metadata was written correctly by extracting it:

    ```bash
    python bmp_manipulator.py metadata extract photo_meta.bmp
    ```

    The output will list every key-value pair stored in the file's metadata section.
  </Step>

  <Step title="Hide a Secret Message">
    Embed a secret text string into a BMP file using LSB steganography:

    ```bash
    python bmp_manipulator.py stego hide photo.bmp \
      --text "This is a secret" \
      --output photo_stego.bmp
    ```

    Retrieve the hidden message at any time with the `stego extract` command:

    ```bash
    python bmp_manipulator.py stego extract photo_stego.bmp
    ```

    The hidden text is stored in the least-significant bits of the pixel data, leaving the image visually unchanged.
  </Step>

  <Step title="Use the Python API">
    You can perform the same operations programmatically by importing BMP Manipulator directly into your Python code:

    ```python
    from bmp_manipulator import BMPFile, Metadata, LSBSteganography

    # Load and inspect
    bmp = BMPFile("photo.bmp")
    info = bmp.get_info()
    print(f"Dimensions: {info['width']}x{info['height']}")

    # Add metadata
    meta = Metadata()
    meta.add("Author", "Alice")
    bmp.add_metadata(meta)
    bmp.save("photo_meta.bmp")

    # Hide text
    stego = LSBSteganography(bmp)
    stego.hide_text("This is a secret")
    bmp.save("photo_stego.bmp")
    ```

    The `BMPFile` class handles all header parsing and file I/O, `Metadata` manages key-value pairs, and `LSBSteganography` encodes or decodes hidden data in pixel bytes.
  </Step>

</Steps>

<Note>
  To protect your hidden data with a password before embedding it, see the [Encryption guide](/guides/encryption). Combining encryption with steganography ensures that even if someone detects the hidden payload, they cannot read it without the correct passphrase.
</Note>

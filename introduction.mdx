BMP Manipulator is a Python library and command-line tool for working with BMP (Bitmap) files beyond basic image editing. It targets the parts of the BMP format that most tools ignore — headers, reserved fields, and unused space — and turns them into a structured environment for metadata management, steganography, and forensic analysis. Whether you need to embed hidden payloads, inspect raw header bytes, or encrypt data inside an image, BMP Manipulator gives you the primitives to do it cleanly from Python code or the terminal.

## Supported BMP Formats

BMP Manipulator supports the three most widely used DIB header variants and three colour depths. Compressed formats (`BI_RLE8`, `BI_RLE4`) and the legacy `BITMAPCOREHEADER` (12 bytes) are **not** supported.

### Header Types

| Header | Size | Notes |
|---|---|---|
| `BITMAPINFOHEADER` | 40 bytes | Most common; baseline support for all features |
| `BITMAPV4HEADER` | 108 bytes | Adds colour-space information; reserved fields available for metadata |
| `BITMAPV5HEADER` | 124 bytes | Adds ICC colour profile offset; maximum reserved field space |

### Colour Depths

| Colour Depth | Description |
|---|---|
| 8-bit indexed (paletted) | Supports palette-manipulation steganography in addition to LSB |
| 24-bit RGB | Primary target for LSB steganography |
| 32-bit RGBA | LSB steganography with alpha-channel support |

## When to Use BMP Manipulator

**Security research and digital forensics.** BMP Manipulator lets you study exactly which header bytes a given tool reads, which it ignores, and what an attacker could hide in the gaps — making it a practical companion for forensic image analysis and CTF challenges.

**Secure data embedding.** When you need to attach provenance information, watermarks, or confidential notes to an image without altering its visible content, BMP Manipulator gives you multiple embedding strategies (header fields, EOF blocks, LSB pixels) with optional AES-256-GCM or ChaCha20-Poly1305 encryption so the data stays private.

**Academic and educational use.** The BMP format's simple, well-documented structure makes it the ideal teaching subject for steganography algorithms and file-format internals. BMP Manipulator's explicit, single-file architecture keeps the implementation readable and easy to extend.

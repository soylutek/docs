Import the classes you need directly from the `bmp_manipulator` module:

```python
from bmp_manipulator import BMPFile, Metadata, LSBSteganography
from bmp_manipulator import MetadataStorageMethod, StegoMethod
```

## BMPFile

The primary entry point for loading, inspecting, modifying, and saving BMP files.

**Constructor**

```python
BMPFile(file_path=None)
```

If you pass `file_path`, the constructor automatically calls `load()`. Omit it to create an empty instance and call `load()` manually later.

### Methods

**`load(file_path: str) -> None`**

Load a BMP from disk and parse all header fields and pixel data.

<ParamField path="file_path" type="str" required>
  Absolute or relative path to the BMP file.
</ParamField>

---

**`get_info() -> dict`**

Return a dictionary describing the file. Keys: `file_path`, `width`, `height`, `bits_per_pixel`, `header_type`, `compression_type`, `pixel_data_offset`, `file_size`, `has_metadata`.

---

**`add_metadata(metadata: Metadata, method: MetadataStorageMethod = EOF_APPEND, password: str = None) -> None`**

Attach a `Metadata` object to the file. The metadata is serialized and written when you call `save()`.

<ParamField path="metadata" type="Metadata" required>
  A populated `Metadata` instance to embed.
</ParamField>
<ParamField path="method" type="MetadataStorageMethod">
  Storage strategy. Defaults to `MetadataStorageMethod.EOF_APPEND`.
</ParamField>
<ParamField path="password" type="str">
  Optional password. When provided, the metadata block is encrypted with AES-256-GCM before storage.
</ParamField>

---

**`extract_metadata(password: str = None) -> Optional[Metadata]`**

Return the embedded `Metadata` object, or `None` if no metadata block is present.

<ParamField path="password" type="str">
  Decryption password. Required when the metadata was stored encrypted.
</ParamField>

---

**`save(output_path: str) -> None`**

Write the BMP — including any attached metadata — to disk.

<ParamField path="output_path" type="str" required>
  Destination file path. The file is created or overwritten.
</ParamField>

### Properties

| Property | Type | Description |
|---|---|---|
| `width` | `int` | Image width in pixels |
| `height` | `int` | Image height in pixels |
| `bits_per_pixel` | `int` | Color depth (8, 24, or 32) |
| `file_size` | `int` | Total file size in bytes |
| `pixel_data_offset` | `int` | Byte offset to pixel data |
| `compression_type` | `str` | Compression method name |
| `header_type` | `str` | DIB header version name |

### Example

```python
bmp = BMPFile("photo.bmp")
info = bmp.get_info()
print(info["width"], info["height"])
```

## Metadata

A key-value store for metadata entries. Create an instance, populate it with `add()`, then pass it to `BMPFile.add_metadata()`.

**Constructor**

```python
Metadata()
```

### Methods

**`add(key: str, value: str) -> None`**

Insert or overwrite a metadata entry.

---

**`get(key: str) -> Optional[str]`**

Retrieve the value for a key, or `None` if the key does not exist.

---

**`remove(key: str) -> None`**

Delete a metadata entry by key.

---

**`to_dict() -> dict`**

Return all entries as a plain Python dictionary.

---

**`to_bytes() -> bytes`**

Serialize the metadata into a binary format suitable for embedding in a BMP file.

---

**`Metadata.from_bytes(data: bytes) -> Metadata`** *(class method)*

Deserialize a binary blob back into a `Metadata` instance. Use this when you retrieve raw bytes from a custom storage location.

### Example

```python
meta = Metadata()
meta.add("Author", "Alice")
meta.add("Date", "2024-01-15")
print(meta.to_dict())
# {'Author': 'Alice', 'Date': '2024-01-15'}
```

## LSBSteganography

Implements Least Significant Bit (LSB) steganography over 24-bit and 32-bit uncompressed BMP files.

**Constructor**

```python
LSBSteganography(bmp_file: BMPFile)
```

<ParamField path="bmp_file" type="BMPFile" required>
  A loaded `BMPFile` instance. The file must be 24-bit or 32-bit uncompressed (`BI_RGB`).
</ParamField>

### Methods

**`calculate_capacity(bit_depth: int = 1, channels: int = 3) -> int`**

Return the maximum number of bytes that can be hidden given the specified parameters.

---

**`hide_data(data: bytes, bit_depth: int = 1, channels: int = 3, password: str = None) -> None`**

Embed arbitrary binary data into the carrier image pixels.

<ParamField path="data" type="bytes" required>
  The payload to hide.
</ParamField>
<ParamField path="bit_depth" type="int">
  Number of least-significant bits per channel to use. Range: 1–4. Default: `1`.
</ParamField>
<ParamField path="channels" type="int">
  Number of color channels to use for embedding. Range: 1–3. Default: `3`.
</ParamField>
<ParamField path="password" type="str">
  Optional password. When provided, the payload is encrypted with AES-256-GCM before embedding.
</ParamField>

---

**`hide_text(text: str, encoding: str = 'utf-8', **kwargs) -> None`**

Convenience wrapper around `hide_data()`. Encodes `text` to bytes using `encoding`, then hides it. All `**kwargs` are forwarded to `hide_data()`.

---

**`extract_data(bit_depth: int = 1, channels: int = 3, password: str = None) -> bytes`**

Read and return the hidden payload from the image. Parameters must exactly match those used during hiding.

---

**`LSBSteganography.extract_text(bmp_path: str, encoding: str = 'utf-8', **kwargs) -> str`** *(static method)*

Load a BMP from `bmp_path`, extract the hidden payload, and decode it as text. All `**kwargs` are forwarded to `extract_data()`.

### Example

```python
bmp = BMPFile("photo.bmp")
stego = LSBSteganography(bmp)

capacity = stego.calculate_capacity(bit_depth=1, channels=3)
print(f"Can hide up to {capacity} bytes")

stego.hide_text("Secret", password="pass")
bmp.save("output.bmp")

text = LSBSteganography.extract_text("output.bmp", password="pass")
print(text)  # "Secret"
```

## MetadataStorageMethod Enum

Controls where metadata is physically written inside the BMP file.

| Value | Description |
|---|---|
| `MetadataStorageMethod.EOF_APPEND` | Append a `BMPM` block after pixel data (default) |
| `MetadataStorageMethod.HEADER_EXTENSION` | Use reserved header fields (limited space) |
| `MetadataStorageMethod.APPLICATION_BLOCK` | Insert an application block |

---

## Exceptions

| Exception | Raised when |
|---|---|
| `BMPError` | File cannot be read or has an unsupported format |
| `MetadataError` | Metadata block is malformed or CRC check fails |
| `SteganographyError` | Payload exceeds capacity or extraction fails |

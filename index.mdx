BMP Manipulator gives you full control over BMP files beyond simple image editing. Use it to inspect every byte of a BMP header, embed custom metadata with integrity checking, or hide text and files inside images using steganography — all with optional AES-256-GCM or ChaCha20-Poly1305 encryption.

## Getting Started

<Steps>
  <Step title="Install">
    Clone the repository and install Python dependencies.
    ```bash
    git clone https://github.com/soyluoltu/bmp-manipulations.git
    cd bmp-manipulations
    pip install -r requirements.txt
    ```
  </Step>
  <Step title="Inspect a BMP file">
    Use the CLI to view header details, dimensions, and metadata status.
    ```bash
    python bmp_manipulator.py info photo.bmp
    ```
  </Step>
  <Step title="Embed metadata">
    Attach a key-value pair to any BMP file.
    ```bash
    python bmp_manipulator.py metadata add photo.bmp \
      --key "Author" --value "Alice" --output photo_meta.bmp
    ```
  </Step>
  <Step title="Hide a secret message">
    Embed text invisibly using LSB steganography.
    ```bash
    python bmp_manipulator.py stego hide photo.bmp \
      --text "Secret message" --output photo_stego.bmp
    ```
  </Step>
</Steps>

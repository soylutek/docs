| | `231` | `+  ### Dosya Başlığı (BITMAPFILEHEADER)  ` |
| | `232` | `+  ` |
| | `233` | `+  | Offset | Boyut | Ad | Açıklama |  ` |
| | `234` | `+  |--------|-------|----------------|-------------|  ` |

| | `236` | `+  | 0x02 | 4 | bfSize | Dosyanın toplam boyutu (bayt) |  ` |
| | `237` | `+  | 0x06 | 2 | bfReserved1 | Rezerve edilmiş (genellikle 0) |  ` |
| | `238` | `+  | 0x08 | 2 | bfReserved2 | Rezerve edilmiş (genellikle 0) |  ` |
| | `239` | `+  | 0x0A | 4 | bfOffBits | Piksel verilerine offset |  ` |
| | `240` | `+  ` |
